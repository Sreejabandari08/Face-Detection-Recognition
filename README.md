# Face-Detection-Recognition
This project is an AI-based Face Detection and Recognition system developed using Python and OpenCV. The system detects human faces from images or webcam video in real-time and can be extended for face recognition and attendance systems.
## Features
- Detects faces from images
- Real-time face detection using webcam
- Uses Haar Cascade Classifier
- Fast and lightweight
- Beginner-friendly Computer Vision project
- Works in Google Colab and VS Code
##  Technologies Used
- Python
- OpenCV
- NumPy
- Matplotlib
- Haar Cascade Algorithm
##  How It Works
1. Input image or webcam video is captured
2. Image is converted to grayscale
3. Haar Cascade classifier scans the image
4. Faces are detected
5. Rectangles are drawn around detected faces
